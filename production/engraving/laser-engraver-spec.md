## Maintenance
- **Field lens**: Clean with lint-free cloth + absolute ethanol when power attenuation occurs
- **Galvanometer mirrors**: Clean with lint-free cloth + absolute ethanol (never touch with hands)

## Maharahjong Production Notes

### Tile engraving (primary use case)
- Tile face is 32 × 23 mm — fits easily within the 200 × 200 mm work area
- At 32 × 23 mm per tile, the work area can fit ~48 tiles per batch (6 × 8 grid with spacing)
- UV cold processing is ideal for engraving on ivory/cream tile material without burning or discoloration
- 0.02 mm laser spot enables the fine rangoli details and Devanagari characters in the tile designs
- Use L-shaped positioning piece for repeatable batch placement
- Export tile designs as SVG for best results (vector line art)

### Brass engraving (dice, trinkets)
- UV laser can engrave brass but **cannot cut metal** — this is a surface marking/engraving machine only
- For dice pip depth (~0.5 mm), multiple passes will be needed
- Brass wind markers and scoring coins: use the batch processing workflow with the L-shaped positioning piece
- Consider the RA3 Rotary for any cylindrical brass pieces

### Wood engraving (racks)
- UV produces clean "frosty" engraving on wood with zero charring
- Ideal for the Maharahjong logo and wind symbol engraving on rack ends
- 15 mm cutting depth on basswood could allow cutting rack blanks, but dedicated woodworking tools are more practical

### Design file preparation
- Prepare all tile designs as SVG (vector paths, not rasterized) for sharpest engraving
- Include registration marks for batch alignment
- Test engraving parameters on sample material before production runs
- Recommended: create an xTool Studio project template for each tile category with pre-set positions and parameters
